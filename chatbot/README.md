# chatbot

## 環境構築(構築済みで不要)

- イメージの作成
  
```bash
docker-compose build
```

- poetryによるPython環境のセットアップ
```bash
docker-compose run \
  --entrypoint "poetry init \
    --name chatbot\
    --dependency fastapi \
    --dependency uvicorn[standard]" \
  chatbot
```


- パッケージインストール
  ```bash
  docker-compose run --entrypoint "poetry install --no-root" chatbot
  ```


- fastAPIの立ち上げ
```
docker-compose up
```
